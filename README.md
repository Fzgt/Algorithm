# Algorithms & JavaScript Internals

Working repository for data structures, algorithms, and from-scratch implementations of JavaScript language and library internals. Written primarily in TypeScript and organised by technique rather than by problem number, so each folder builds one transferable pattern instead of a list of one-off answers.

Actively maintained since February 2025.

## Contents

**Algorithms** — ~170 solutions across 13 topics: linked lists, binary trees, dynamic programming, backtracking, sliding window & two pointers, graphs, binary search, stack & queue, hash tables, greedy, and sorting. The backtracking and graph topics include written breakdowns with recursion traces alongside the code.

**JavaScript internals** — from-scratch implementations of `Promise` (full A+ spec: thenable resolution, cycle detection, single-call guard), `call` / `apply` / `bind`, `new`, `instanceof`, `Object.create`, deep clone, debounce / throttle, currying, pub-sub, and the observer pattern.

**Applied problems** — larger exercises: time slicing, npm circular dependency detection, list-to-tree transforms, and batched request scheduling.
